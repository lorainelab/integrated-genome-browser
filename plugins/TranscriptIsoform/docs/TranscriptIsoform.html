Plugin Name: TranscriptIsoform

Version: 0.0.9 (BETA release)

(Designed to work with IGB6.7 and up)

Date: 2011-06-15

Author: Genentech

Overview:    BETA release of transcript isoform evidence visualization.
  
Usage:The TranscriptIsoform plugin is an implementation of the TierMaintenanceListener extension point.
The Transcript Isoform Evidence plugin displays evidence of exon-exon junctions from RNA-seq read alignments
that span those junctions. An observed junction is indicated by a chevron (/\), the line thickness of which
is derived from the number of supporting reads. The color indicates whether the junction exists in the loaded
gene models (blue) or is a novel junction (red).
